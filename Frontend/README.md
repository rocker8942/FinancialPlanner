# Financial Planner Frontend

This is the frontend for the Financial Planner web application, built with Vue 3, TypeScript, and Vite.

## Features
- User authentication (login/register)
- Input and visualization of assets, salary, and expenses
- Interactive net wealth chart
- Profile and settings management
- Responsive design

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Run the development server:
   ```sh
   npm run dev
   ```

## Project Structure
- `src/` - Main source code
- `src/views/` - Page components
- `src/components/` - Reusable UI components
- `src/store/` - Pinia stores
- `src/router/` - Vue Router setup

## Linting & Formatting
- ESLint and Prettier are included for code quality.

## API
- Connects to the .NET Core backend via REST API.
